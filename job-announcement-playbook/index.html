Release notes

Release 6.6

This release introduces new layout and content guidelines for job announcements. The focus of the release is on the job announcement however, there a few other changes of note:

New hiring path: Internal to an agency

We heard your feedback and have added a new hiring path: Internal to agency. This option should be used to indicate that a job is only open to employees of the posting agency. Using the text “agency only” in the Who May Apply section will cause the job announcement to be tagged with this hiring path.

There is also a new page in our Help Center for this hiring path: https://www.usajobs.gov/Help/working-in-government/unique-hiring-paths/federal-employees/internal/

Land management hiring path demoted

The Land Management hiring path was removed from search filtering, job announcements (section “This job is open to…”), and the landing page. A page remains in the Help Center dedicated to explaining this hiring path to applicants. It is now found at this URL:

https://www.usajobs.gov/Help/working-in-government/unique-hiring-paths/federal-employees/land-management/

This was done because a land management eligible applicant can actually apply to the same positions as any other merit promotion eligible federal employee. Thus there was no need to have a distinct filter for search which is a main criteria for creating and maintaining a hiring path.

Base management hiring path sub-page created

While we nested the Land Management hiring path page under Federal employees we also added a page to highlight a similar eligibility for employees of the Defense Industrial Base Facility or the Major Range and Test Facilities Base. That new page can be found here:

https://www.usajobs.gov/Help/working-in-government/unique-hiring-paths/federal-employees/base-management/
